## Risk Notes

- **Windows local locking is no-op**: `flock_other.go` provides no cross-process safety on Windows. Acceptable for current usage (darwin+linux CI/dev).
- **Local dependency waiting returns error, not poll**: When `DepsWaiting=true`, the local `BeforeJob` hook returns an error rather than polling. This is acceptable for the default single-process topological runner (never hits this path). Multi-process local usage with `--job` requires the caller to retry.
- **Remote heartbeat timeout**: Not deeply exercised by the short conformance fixture (jobs finish in seconds). The 5-minute heartbeat timeout is tested only structurally.
- **Remote logs are job-level aggregated**: The CLI displays flattened step output rather than per-step segmented view. UX polish item, not correctness.
- **Node.js 20 deprecation warning**: GitHub Actions annotations warn that `actions/checkout@v4`, `actions/setup-go@v5`, and `actions/upload-artifact@v4` will need updates before September 2026.
